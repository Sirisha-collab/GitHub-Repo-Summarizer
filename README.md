# GitHub-Repo-Summarizer
Scans only public GitHub repositories, builds a complete project structure, detects architecture, and generates summary using Large Language Models.

## ✨ Features

- 📂 Analyze local folders or GitHub repositories
- 🔄 Clone public GitHub repositories automatically
-  Generate a repository file tree
- 📦 Detect:
  - Imports
  - Classes
  - Functions
  - Methods
  - Entry points
  - External dependencies
- ⚡ Support Groq API (fast)
- 🤗 Support Hugging Face Inference API

## 📦 Installation
